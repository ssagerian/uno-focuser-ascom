\# Uno ASCOM Focuser



Arduino-based telescope focuser controlled via ASCOM local server driver.



\## Features

\- Absolute positioning

\- Serial protocol over USB

\- Compatible with ASCOM Device Hub / NINA



\## Status

Working prototype



\## to install

unzip release.zip into folder C:\\Program Files (x86)\\ASCOM\\ascom\_driver

run ASCOM.ssagerianunofocuser.exe /regserver

verify in NINA
